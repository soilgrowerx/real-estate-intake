To run comparables effectively, I need the following information:

1. **Lot Address or Parcel ID**: The specific location of the lot helps identify relevant comparables in the area.
2. **Lot Size**: Precise acreage or size range is essential to narrow down similar properties.
3. **Lot Use Code**: Whether the land is residential, agricultural, commercial, etc., determines the type of comparables to consider.
4. **Utilities Information**: Whether utilities like water, sewer, and electricity are available or nearby.
5. **Access Information**: If the lot has road frontage or is accessible.
6. **Additional Features**: Information about the lot&#x27;s topography, proximity to developments, flood zones, or wetlands.

If you provide any combination of these, I can begin researching and evaluating the property for comparables.
