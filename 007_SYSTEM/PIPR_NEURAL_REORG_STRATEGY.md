## 🚀 Neural Layer (The Master Move)
- **Flattening:** Goose will pull files from deep sub-folders (e.g., `RESOURCES/Ecology/Biomimicry/file.md`) into the top-level `004_RESOURCES/`.
- **Linking:** Every moved file gets a tag based on its origin (e.g., `#domain/biomimicry`).
- **MOCs:** Pi will generate a `Forestry MOC.md` and `Ecology MOC.md` in `007_SYSTEM/MOCS/` to provide high-level entry points.
