# SKILL.md - Self-Improving Agent

## Purpose
Enable the agent to learn from errors, user corrections, and successful execution patterns to optimize the Neocambrian workflow.

## Instructions
1.  **Error Harvesting:** When a tool call fails, analyze *why* and document the fix.
2.  **User Correction:** Treat user feedback as "Ground Truth" updates to operational strategy.
3.  **Pattern Recognition:** Identify repetitive manual steps (like re-auth commands) and automate or document them for permanent recall.
4.  **Skill Evolution:** Periodically review local `.md` files in the `skills/` directory to refine instructions.
